## Using Watch Mode & VScode liveServer

    -- package.json --

    "scripts": {
        "watch": "webpack --watch",
        "build": "webpack"
    },

사실 나머지 방법의 필요성을 못느끼겠음.

- react 를 쓸 때도,
- spa 가 아닌 여러 페이지로 이뤄진 경우에도,
- vanila js 로 spa 개발할 때도

  아무 문제 없어보임.
  공식사이트에서는 아마 live Server 같은 확장 프로그램을 쓰지 않는다고 가정한듯.
